# Nihongogo - 实用场景日语表达训练系统

---

## 产品定位

Nihongogo 是专为中文母语学习者打造的日语实用场景输出训练平台。不同于传统的词汇记忆和考级导向课程，Nihongogo 聚焦于真实场景下的长句听力理解与口语表达能力，帮助用户快速在日常生活、职场沟通、社交互动中流畅表达。

---

## 核心训练目标

* **长句听力理解与实时反应能力**
* **复杂语句组织与流畅口语输出能力**
* **场景应用表达与文化逻辑迁移能力**

---

## 产品特色

### 1. 双导向学习路径设计

* **主导向：系统化训练营**：90天快速突破、30天职场专项、14天旅游应急等完整学习路径
* **辅助导向：模块化强化训练**：听力训练、口语练习、场景对话等针对性技能提升
* **智能路径推荐**：根据用户水平和目标智能推荐最适合的学习入口
* **无缝模式切换**：训练营学习中可随时进入模块强化，模块训练后可升级到系统课程
* **个性化学习中心**：统一的学习仪表板，支持两种学习模式的进度管理和效果追踪

### 2. 综合听力与跟读训练模块

* 提供真实场景的长句听力素材
* 支持语速调节和关键词高亮功能
* 提供逐句拆解、精细听写和复述训练功能
* Shadowing 模拟跟读训练与实时发音反馈
* 听力理解到口语输出的一体化训练流程

### 3. AI智能对话训练

* AI支持的表达组织训练
* 场景模拟多轮交互对话
* 智能对话引导与纠错反馈
* 复杂语境下的表达能力训练

### 4. 表达组织能力训练引擎

* 训练复杂句式逻辑组织能力（并列、转折、假设、总结）
* 丰富的表达模板库供学员练习
* 场景内化长句表达训练

### 5. 文化表达习惯与迁移训练

* 场景表达习惯差异化教学
* 日语特有表达逻辑与习惯的教学
* 敬语、委婉表达、场景文化内化训练

### 6. AI智能学习辅助工具

* 学习瓶颈识别与定制化提升训练
* 用户成长曲线动态展示与能力诊断
* 智能化词汇和语法复习引擎（艾宾浩斯遗忘曲线）

### 7. 高效内容生产与管理体系

* **可视化课程配置器**：拖拽式训练营设计，无需编程即可创建复杂学习路径
* **模块化内容库管理**：课程模块、场景素材、评测标准的统一管理和复用
* **智能课程生成引擎**：基于目标和时间自动生成最优学习路径
* **多维度配置支持**：时间安排、难度梯度、技能重点、文化背景等全方位定制
* **A/B测试框架**：支持不同课程配置的效果对比和优化迭代
* **AI辅助内容批量生成**：标准化数据模型，支持多语言内容扩展

## 核心数据结构设计

### 🎯 以句子为核心的解耦设计

Nihongogo 采用以句子为核心单元的数据结构设计，实现内容与逻辑的完全解耦，支持灵活的多语言扩展和课程配置。

**核心设计优势：**
- ✅ **最大化复用性** - 一个句子可用于多个场景和训练模式
- ✅ **精细化控制** - 可针对单个句子进行详细的训练配置
- ✅ **灵活组合** - 管理端可自由组合句子构建课程
- ✅ **数据驱动** - 可分析到句子级别的学习效果
- ✅ **易于扩展** - 新增语言或训练模式时结构不变
- ✅ **AI友好** - 结构化数据便于AI分析和生成

#### 语言处理引擎接口
```typescript
interface LanguageProcessor {
  languageCode: string;

  // 文本处理
  tokenize(text: string): Token[];
  analyzeMorphology(text: string): MorphologyAnalysis;
  extractKeywords(text: string): Keyword[];

  // 发音处理
  generatePronunciationGuide(text: string): string;
  getPhonemicTranscription(text: string): Phoneme[];

  // 语言特定功能
  getLanguageSpecificFeatures(text: string): LanguageFeatures;
  validateGrammar(text: string): GrammarValidation;
}

// 日语处理器实现
class JapaneseProcessor implements LanguageProcessor {
  languageCode = 'ja';

  tokenize(text: string): Token[] {
    // 使用MeCab或类似工具进行日语分词
    return this.mecabTokenizer.parse(text);
  }

  generatePronunciationGuide(text: string): string {
    // 生成罗马字读音
    return this.romajiConverter.convert(text);
  }

  getLanguageSpecificFeatures(text: string): JapaneseFeatures {
    return {
      keigoLevel: this.detectKeigoLevel(text),
      particleUsage: this.extractParticles(text),
      kanjiReadings: this.getKanjiReadings(text)
    };
  }
}

// 韩语处理器实现
class KoreanProcessor implements LanguageProcessor {
  languageCode = 'ko';

  tokenize(text: string): Token[] {
    // 使用KoNLPy或类似工具
    return this.konlpyTokenizer.parse(text);
  }

  getLanguageSpecificFeatures(text: string): KoreanFeatures {
    return {
      honorificLevel: this.detectHonorificLevel(text),
      verbEndings: this.extractVerbEndings(text),
      particleUsage: this.extractParticles(text)
    };
  }
}
```
